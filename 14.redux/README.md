# Music Player

Using just https://redux.js.org/ create a nodejs console application.


Working Example:

```
> loadPlaylist example.json
Play list Loaded
> play
playing
> status
playing - Eminem - Sing For The Moment
> stop
stopped
> status
stopped - Eminem - Sing For The Moment
> playNextSong
> status
playing - Eminem - Lose Yourself 
```
